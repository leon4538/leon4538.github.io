Introduction

HTML stands for HyperText Markup Language and it is the standard markup language for documents designed to be displayed in web browsers. Markup languages refer to computer languages that consist of keywords, names, or tags to control its structure and relationships.

Within the HTML language, there is another used to describe the presentation of a document. Cascading Style Sheets (CSS) is used to style and layout web pages. For example:

  • Altering fonts, size, color, and spacing of content.
HTML Basics

HTML consists of a series of elements that are used to describe the structure of a web page. Elements within this language tell the browser how to display its content.

Here is the very basic setup for a simple HTML document:

  • The <!DOCTYPE html> declaration defines the document type. Helping the browser to display the page correctly.
  • The <html> element is the root element of an HTML page
  • The <head> element contains meta information about the HTML page
  • The <title> element specifies a title for the HTML page (shown in the browser's title bar or tab)
  • The <body> element defines the document's body, and is the visible part of a document.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph

HTML Elements

An HTML element is defined by everything within a start and end tag. Typically, it is defined as a start tag, some content, and an end tag.

Here is an example:

<tagname> Content goes here... </tagname>

CSS Basics

CSS stands for Cascading Style Sheets and is used to format the layout of a webpage. You can control the color, font, text-size, displays for different devices, and more.

CSS can be added to an HTML document in 3 different ways:

  • Inline: by using the style attribute inside HTML elements
  • Internal: using a <style> element in the head section.
  • External: using a <link> element to link an external CSS file.

With an external style sheet, you can change the way a website looks by changing one file.

Common CSS Properties

Here are some commonly used properties to add styling to a web page:

  • color : defines text color
  • font-size : defines the text size
  • background-color : defines background color for an element
  • border : defines a border around an HTML element
  • padding : adds space (padding) between text and its border.
